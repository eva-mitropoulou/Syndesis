# Stage 8 Candidate Screening Protocol

Stage 8 applies the validated EGFR docking cascade to Stage 7 molecules. It uses receptor-ensemble docking, GNINA rescoring, ProLIF interaction fingerprints, the saved Stage 6 pose-confidence model, and molecule-level aggregation.

The default config runs a bounded smoke/control screen because Stage 7 currently contains known controls but no successful prospective vendor/generated/manual imports.
