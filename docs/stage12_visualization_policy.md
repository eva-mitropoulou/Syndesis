# Stage 12 Visualization Policy

Figures are generated from existing tables only. They must not imply experimental activity.

Required figure classes include pipeline overview, candidate funnel, source composition, score distributions, pose-confidence versus GNINA summaries, interaction recovery, MD evidence status, analog acceptance, score-hacking examples, 2D structure panels, pose panels, risk heatmaps, and evidence matrices.

2D depictions use RDKit when available. Missing optional visual evidence must be explicit in candidate cards or warnings.
