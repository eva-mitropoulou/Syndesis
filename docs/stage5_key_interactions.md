# Stage 5 Key Interactions

Key EGFR interactions are selected from native co-crystal interaction frequency plus configured manual EGFR kinase roles.

Residue roles include Lys745, Glu762, Thr790, Met793, Cys797, Asp855, Phe856, and Gly857. KLIFS positions are used when present; UniProt numbering is the fallback.
