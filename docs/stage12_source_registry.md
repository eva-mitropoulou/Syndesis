# Stage 12 Source Registry

The machine-readable Stage 12 source registry is `data/references/stage12_sources.yaml`.

It covers FAIR reporting, model cards, datasheets for datasets, reproducible software reporting, reproducible biomedical notebooks, ProLIF, PLIP, Mol*, GNINA, PoseBusters, interaction recovery, ultra-large docking, RDKit, and SHAP.

Each source entry includes a supported claim, bibliographic metadata, DOI or URL, how it is used, and confidence level.
