# Stage 11 MD Protocol

Stage 11 is an explicit-solvent all-atom MD stress test for finalist EGFR poses. The default stack is GROMACS with CHARMM36m protein, CGenFF ligand, CHARMM TIP3P water, and CHARMM-compatible ions at 0.15 M NaCl.

This stage does not run FEP, metadynamics, accelerated MD, umbrella sampling, absolute binding free energy, or ligand-unbinding simulations.
